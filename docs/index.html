Strategy Description

Entry Rules

  • Bollinger Band Entry: Buy when price drops below BB lower band (-2.0σ)
  • LSTM Filter: Only enter if LSTM predicts hold time < 3 days (good entry)
  • DCA: Split into 2 entries with 5% avg-down threshold

Exit Rules

  • Take Profit: Exit when price rises above BB upper band (+1.0σ)
  • Time Limit: Force exit after 60 days max hold
  • No Stop Loss: Position held until TP or time limit

LSTM Model

  • Input: 100-candle OHLCV sequence + 16 technical features
  • Architecture: Bidirectional LSTM (64 hidden, 2 layers) + FC classifier
  • Label: hold_days > 3 → bad entry (0), else good entry (1)
  • Class Weights: Applied to handle imbalanced data

Insights

  • 1h timeframe benefits most from LSTM filter - All 4 coins showed improvement at 1h
  • ETHUSDT 1h achieved profit - Only case where LSTM filter turned loss into profit (+6.42%)
  • Short timeframes (5m) show minimal difference - Too much noise for LSTM to add value
  • 2026 Q1 was a bear market - BTC dropped 17%, making all DCA strategies challenging
  • LSTM reduces trade frequency at 1h - More selective entries lead to better results
